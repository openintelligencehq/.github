# Security Policy

We take security seriously and appreciate responsible disclosure.

## Reporting a Vulnerability

Please report vulnerabilities by emailing: **security@openintelligence.in**

Include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested remediation (if available)

## Response Process

- We will acknowledge receipt as quickly as possible.
- We will investigate, validate, and prioritize remediation.
- We may coordinate disclosure timing with the reporter when appropriate.
